Colors

Provided are the built-in color scales, and our gray scale. Each of these colors is defined with a base color class and an optional dark class.

Background Colors

To use as a background color, pick a color value as a class.

This is a card-panel with a teal class. 
                    

<div class="b-card-panel teal z-depth-1">
    This is a card-panel with a teal class
</div>

Text Colors

To use as a text color, pick a color value as a class, and append a -text.

This is a card-panel with a teal-text class. 
                    

<div class="b-card-panel teal-text z-depth-1">
        This is a card-panel with a teal class.
</div>

Typography

The standard font soshBuilder uses is Benton Sans.

Benton Sans has become popular among newspaper and magazine publications because of its uncomplicated, straightforward structure. Because of its ability to function as both body copy and in larger display sets, Fortune magazine implemented it into its 2007 redesign.

Benton Sans can also be seen in the Houston Chronicle and many other publications. Global design firm, frog design, uses Benton Sans in all its published materials. Additionally, the New York Art Directors Club has used the typeface family for many of its branding and promotional materials.

Flow Text

One common flaw seen in many frameworks is a lack of support for truly responsive text. While elements on the page resize fluidly, text still resizes on a fixed basis. To ameliorate this problem, for text heavy pages, supplied is a class that fluidly scales text size and line-height to optimize readability for the user. Line length stays between 45-80 characters and line height scales to be larger on smaller screens. 
                    

<div class="flow-text b-card-panel z-depth-2">
    ... your flowing text would go here ...
</div>
